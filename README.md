# 黑塔转圈圈

> 本项目由[阿里云ESA](https://www.aliyun.com/product/esa)提供加速、计算和保护<a href="https://www.aliyun.com/product/esa"><img src="https://img.alicdn.com/imgextra/i3/O1CN01H1UU3i1Cti9lYtFrs_!!6000000000139-2-tps-7534-844.png" width="250"></a>

“黑塔转圈圈” 是前端初学者基于 AI 辅助打造的网页小游戏，以热门 IP 角色 “黑塔” 转圈圈计数为核心玩法，搭配角色银狼的吐槽反馈，互动简单直观。

项目基于 CSS 动画、TailwindCSS、Vue.js、Astro 等技术实现，并通过阿里云 ESA Pages 实现快速部署与访问。
